2.0 Getting Started Tutorial
________________________________________________________________________________________

	This will be a brief tutorial on how to get setup with development in
	the nodejs environment. A more detailed tutorial can be found on our
	trello board. https://trello.com/c/lAobEJQ9/29-full-tutorial

	1.	Download and install Node.js. https://nodejs.org/

	2.	Download and install Github for Windows 
		https://windows.github.com/ (or of course just use GIT).
		
	3.	Contact a member of the "CSSEBuddySystem" to invite you to
		the project. Then in Github for Windows click the "+" icon
		in the top left corner, then the "Clone" tab, then the 
		"CSSEBuddySystem" organization and then the "CS-BS-Website".

	4.	Run the "StartServer.bat" in the "bat" directory and go to
		"http://localhost:3000/" to view a local instance of the 
		project.
	
	5.	(Optional) If you want to install a new package or
		"StartServer.bat" isn't working, run "InstallPackages.bat"
		to reinstall all the packages listed in "package.json".
